# Introduction #
The FantaBobMod is a *DISCONTINUED* Minecraft mod based on the Minecraftian celebrities TheFantasio974 and Boblennon.  
It adds multiple items, blocks, achievements and mods to the game.  
The last version published is for Minecraft 1.3.2, and it will hopefully be updated and rewritten when the official modding API is released.

# Installation #
Install ModLoader inside minecraft.jar. Copy the resources folder inside .minecraft. Copy the rest in minecraft.jar.
